evol.colorpicker demo

GitHub

evol.colorpicker (currently v3.1.0) is a web color picker which looks like the one in Microsoft Office 2010. It can be used inline or as a popup bound to a text box. It is a full jQuery UI widget, supporting various configurations and themes.

Themes: ui-lightness - ui-darkness - redmond - sunny - le-frog
