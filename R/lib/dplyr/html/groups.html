groups {dplyr}R Documentation

Return grouping variables

Description

group_vars() returns a character vector; groups() returns a list of symbols.

Usage

groups(x)

group_vars(x)

Arguments

x

A tbl()

Examples

df <- tibble(x = 1, y = 2) %>% group_by(x, y)
group_vars(df)
groups(df)
