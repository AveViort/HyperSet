is_some_class {dendextend}R Documentation

Is the object of some class

Description

Returns TRUE if some class (based on the name of the function).

Usage

is.hclust(x)

is.dendrogram(x)

is.phylo(x)

is.dendlist(x)

is.dist(x)

Arguments

x

an object.

Value

Returns TRUE if some class (based on the name of the function).

Examples

# TRUE:
is.dendlist(dendlist())
# FALSE
is.dendlist(1)
# TRUE:
is.dist(dist(mtcars))
# FALSE
is.dist(mtcars)
