with_par {withr}R Documentation

Graphics parameters

Description

Temporarily change graphics parameters.

Usage

with_par(new, code, no.readonly = FALSE)

local_par(new, no.readonly = FALSE, .local_envir = parent.frame())

Arguments

new

[named list]
New graphics parameters and their values

code

[any]
Code to execute in the temporary environment

no.readonly

[logical(1)]
see par() documentation.

.local_envir

[environment]
The environment to use for scoping.

Value

[any]
The results of the evaluation of the code argument.
