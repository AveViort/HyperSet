window_order {dbplyr}R Documentation

Override window order and frame

Description

Override window order and frame

Usage

window_order(.data, ...)

window_frame(.data, from = -Inf, to = Inf)

Arguments

.data

A remote tibble

...

Name-value pairs of expressions.

from, to

Bounds of the frame.

Examples

library(dplyr)
df <- lazy_frame(g = rep(1:2, each = 5), y = runif(10), z = 1:10)

df %>%
  window_order(y) %>%
  mutate(z = cumsum(y)) %>%
  sql_build()

df %>%
  group_by(g) %>%
  window_frame(-3, 0) %>%
  window_order(z) %>%
  mutate(z = sum(x)) %>%
  sql_build()
