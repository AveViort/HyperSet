last_check {rhub}R Documentation

The last rhub check of this R session

Description

rhub caches the id(s) of the last submission. This can be retrieved with last_check.

Usage

last_check()

Value

An rhub_check object.

Examples

## Not run: 
check("packagedir")
last_check()
last_check()$livelog()

## End(Not run)
