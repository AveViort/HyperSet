Has attribute or name?

Usage

has_attr(x, which)

has_name(x, which)

Arguments

x

which

name or attribute

Examples

has_attr(has_attr, "fail")

x %has_attr% "a"

y <- list(a = 1, b = 2)
see_if(y %has_name% "c")
