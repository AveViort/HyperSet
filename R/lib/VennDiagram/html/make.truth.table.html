make.truth.table {VennDiagram}R Documentation

Make a truth table

Description

Makes a truth table of the inputs.

Usage

make.truth.table(x)

Arguments

x

A short vector.

Value

A data frame with length(x) logical vector columns and 2 ^ length(x) rows.

Author(s)

Richard Cotton
