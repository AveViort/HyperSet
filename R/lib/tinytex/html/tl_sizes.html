tl_sizes {tinytex}R Documentation

Sizes of LaTeX packages in TeX Live

Description

Use the command tlmgr info --list --only-installed to obtain the sizes of installed LaTeX packages.

Usage

tl_sizes(show_total = TRUE)

Arguments

show_total

Whether to show the total size.

Value

A data frame of three columns: package is the package names, size is the sizes in bytes, and size_h is the human-readable version of sizes.
