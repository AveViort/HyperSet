list2hexList {hexbin}R Documentation

Convert list to hexList

Description

Converts a list of hexbin objects with same xbnds, ybnds, shape and xbins to a hexList object.

Usage

list2hexList(binlst)

Arguments

binlst

A list of hexbin objects

Value

a hexList object
